Voici comment lancer le programme :

  1 - Téléchargez les différents fichiers et placez-les dans un espace dédié.
  2 - Ouvrez un terminal depuis ce dossier.
  3 - Dans le terminal, entrez la commande "make".
  4 - Effectuez la commande : "./prog".
 
Contenu des fichiers : 

  main.c => programme principal contenant la déclaration des variables, le menu principal ainsi que les boucles de jeu
  fonction.c => programme contenant les différentes fonctions et procédures du programme
  game_board.c => programme contenant la construction du plateau de jeu et les tours de jeu
  header.h => programme contenant l'inclusion des bibliothèques requises, les noms de fonctions, les noms de
  procédures, les structures ainsi que les énumérations
  Makefile => fichier contenant la liste des commandes à exécuter par le terminal lorsque l'utilisateur entre la 
  commande "make".
